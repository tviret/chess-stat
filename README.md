 # Chess-stat
  Chess-stat est une plateforme web qui permet de visualiser des données sur des tournois des parties et des joueurs d'echecs.
## Installation du projet
Le projet s'installe et se démarre simplement via git clone et docker-compose

    git clone https://github.com/tviret/chess-stat
    docker compose up

Il convient de bien laisser le temps aux 3 services de se lancer avant de tester l'application.

Une fois les services lancés on accède au front via

    http://localhost:5173
